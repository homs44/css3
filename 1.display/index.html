Block Element

  • 항상 새로운 라인에서 시작
  • 화면의 가로 전체 크기를 차지한다
  • width, height, margin, padding 프로퍼티 지정이 가능하다.
  • 안에 Block 및 Inline 요소를 포함할 수 있다.

Elements

  • div
  • h1 ~ h6
  • p
  • ol
  • ul
  • li
  • hr
  • table
  • form

Inline Element

  • 새로운 라인에서 시작하지 않고 문장 및 다른 요소들과 함께 같은 행에 위치한다.
  • Content의 너비 만큼 가로폭을 차지한다.
  • width, height, margin-top, margin-bottom 프로퍼티를 지정할 수 없다. 상, 하 여백은 line-height로 지정한다.
  • inline 요소 뒤에 공백이 있는 경우, 공간에 4px가 자동 지정된다
  • inline 요소 안에 block 요소가 들어올 수 없다.
  • span
  • a
  • strong
  • img
  • br
  • input
  • select
  • textarea
  • button

Inline Block Element

  • Inline 요소 처럼 줄을 바꾸지 않고 한 행에 위치 시킨다.
  • block 레벨 요소처럼 width, height, margin, padding 프로퍼티를 모두 정의할 수 있다.
  • content의 너비만큼 가로폭을 차지한다.
  • inline-block 레벨 요소 뒤에 공백이 있는 경우, 공간에 4px가 자동 지정된다
